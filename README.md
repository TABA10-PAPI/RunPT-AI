# RunPT-AI 🏃‍♂️🔋
러닝 기록(거리·페이스·시간·평균 심박)과 사용자 정적 특성(나이·키·몸무게)을 기반으로  
**컨디션/피로도(Battery Score, 0~100)를 예측**하고, 오늘 상태에 맞는 **러닝 추천(Easy/Tempo/Intervals 등)**을 제공하는 **시계열 AI API**입니다.

- **API**: FastAPI
- **Model**: LSTM 기반 배터리 예측 (기본 모델 + 사용자 데이터 기반 개인화 fine-tuning)
- **Data**: 사용자 러닝 기록(JSON) 저장 및 누적 → skill 업데이트 → 배터리/추천에 반영

---

## ✨ 주요 기능
- **배터리 점수 예측**: 최근 7일(WINDOW_SIZE=7) 러닝 이력 + 사용자 프로필 기반 Battery Score 산출
- **점수 설명/피드백 제공**: 휴식일/최근 강한 훈련 여부/피로도 지표를 근거로 이유와 피드백 제공
- **러닝 추천 생성**: 배터리 점수 + acute fatigue + consistency 등을 반영한 추천(최소 3개 보장)
- **개인화 모델 지원**: 사용자 기록이 충분할 때(기본 조건: 20개 이상) 사용자 전용 모델로 예측
